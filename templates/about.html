Neural Network Architectures Used

CRAFT (Text Detection)

Character Region Awareness for Text Detection

  • Detects individual character regions
  • Links characters to form words
  • Works well with curved/rotated text
  • VGG16-based feature extraction

DBNet (Text Detection)

Differentiable Binarization Network

  • ResNet-50 backbone
  • Differentiable binarization module
  • Fast and accurate
  • Better for document layouts

CRNN (Text Recognition)

Convolutional Recurrent Neural Network

  • CNN extracts visual features
  • RNN (LSTM) models sequences
  • CTC loss for training
  • Handles variable-length text

Key Terms Glossary

OCR (Optical Character Recognition)

Technology that converts images of text into machine-readable text data.

CNN (Convolutional Neural Network)

A type of neural network designed for processing grid-like data such as images. Uses convolution operations to extract features.

RNN (Recurrent Neural Network)

A neural network designed for sequential data. Has memory to process sequences of inputs.

LSTM (Long Short-Term Memory)

A type of RNN that can learn long-term dependencies. Used in CRNN for text sequence modeling.

CTC (Connectionist Temporal Classification)

A loss function used to train networks on sequence-to-sequence problems without requiring aligned data.

Pre-trained Model

A model trained on a large dataset that can be used directly or fine-tuned for specific tasks.
